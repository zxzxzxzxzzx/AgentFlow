# AgentFlow

> 通用智能体执行框架——适用于任何领域的任务自动化。

## 是什么

AgentFlow 是一个 Claude Code Skill，将 Claude 塑造为自动完成 **"提示词优化 → 制定计划 → 执行与自适应"** 的通用任务处理者。无论你请求的是写作、编程、分析、策划还是学习，它都按同一套严谨流程推进。

## 三阶段流程

| 阶段 | 做什么 | 核心原则 |
|------|--------|---------|
| **一：提示词优化** | 从模糊请求中提取目标、约束、边界，重写为可执行任务声明 | 缺信息精准回问，不脑补 |
| **二：制定计划** | 领域无关步骤拆解，每步含行动描述+所需能力+完成标准 | 禁止占位符，每步可验证 |
| **三：执行与自适应** | 严格顺序推进，异常分级处理，受阻时动态调整 | 3 次失败即质疑计划本身 |

## 特点

- **领域中立**：不偏向编程、写作或任何特定领域
- **复杂度自适应**：简单任务静默执行，复杂任务逐阶段确认
- **门控机制**：每个阶段有硬性退出标准，不达标准不进入下一阶段
- **反合理化设计**：预判 AI 可能走的捷径，逐条封堵

## 安装

```bash
npx skills add Thezx-a/AgentFlow --skill AgentFlow
```

## 使用

直接向 Claude 提出任何任务请求，AgentFlow 自动激活：

```
帮我写一份市场分析报告
```

```
设计一个 REST API 的用户认证系统
```

```
帮我规划三个月的深度学习自学路线
```

需要跳过确认环节时，在请求末尾加上"自动执行"即可。
